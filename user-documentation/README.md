#### ADDITIONAL INFORMATION AND PRECAUTIONS

- IF THE FIRST IMAGE THE MODEL GENERATES IS CLOSE BUT NEEDS MODIFICATION, DO SO WITH TEXT PROMPTS. ALWAYS MAKE YOUR TEXT PROMPTS CLEAR, SENSIBLE, AND ENSURE ACCURATE SPELLING AND GRAMMAR

- IF AFTER AN ATTEMPT AT EDITING THE ORIGINAL IMAGE FAILS, ADDITIONAL STEPS MAY BE TAKEN TO ACHIEVE DESIRED RESULTS:

    - CREATE A NEW THREAD AND MIGRATE THE "BEST" GENERATED IMAGE FROM THE PREVIOUS THREAD TO THE NEW THREAD. DESCRIBE THE DESIRED CHANGES
    - CREATE A NEW THREAD AND PROVIDE THE MODEL WITH THE ORIGINAL DOCUMENTS/IMAGES. AMEND THE TEXT PROMPT TO REFLECT THE DESIRED OUTCOME WITH EDITS

- IN THE EVENT A GENERATION FAILS MULTIPLE TIMES, DO NOT CONTINUE TO TRY TO MODIFY THE IMAGE IN THE CURRENT THREAD. THE MODEL HAS ALREADY INGESTED THE USER-PROVIDED CONTEXT, AND MAY NOT BEHAVE AS DESIRED

- THE MODEL INTERPRETS NATURAL LANGUAGE EXTREMELY WELL, BUT ALWAYS (!!!) ENSURE TEXT PROMPTS ARE WELL-STRUCTURED, THOUGHTFUL, COMPLETE, AND WELL-WRITTEN
